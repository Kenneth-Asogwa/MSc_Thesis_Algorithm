Project Title
AI-Powered Payroll Fraud Detection in Public Financial Management

Overview
This repository contains the implementation of a deep learning framework for detecting payroll fraud in Nigeria’s Public Financial Management (PFM) system. The model uses Convolutional Neural Networks (CNN) combined with SMOTE to address class imbalance and detect anomalous payroll transactions.

Key Features
1. Payroll fraud detection using CNN

2. Handling imbalanced data using SMOTE

3. Model evaluation using ROC, Precision, Recall, F1-score

4. AUC = 0.94

5. Integration into a real-time fraud monitoring dashboard

The dataset used in this study is a simulated Nigerian payroll dataset designed to reflect real-world characteristics of public sector payroll systems.
